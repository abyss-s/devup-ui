# Box

The `Box` component is a layout primitive that can be used to create any kind of layout.

It is just a `div` with some styles.

## How to use

```jsx
// Before
<Box bg={"red"} />

// After
<div className="d0" />
```

You can use the `as` prop to change the element type.

```jsx
// Before
<Box as="span" />

// After
<span className="d0" />
```
