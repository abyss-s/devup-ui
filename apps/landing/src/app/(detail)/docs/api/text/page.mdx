# Text

The `Text` component is a layout primitive that can be used like `span` to create texts.

It is just a `span` with some styles.

## How to use

```tsx
import { Text } from '@devup-ui/react'

function App() {
  return (
    <Text color="red" fontSize="24px">
      This is Text component.
    </Text>
  )
}
```

The Text component defined above will render like this:

```tsx
function App() {
  return <span className="aa ab">This is Text component.</span>
}
```

```css
.aa {
  color: red;
}
.ab {
  font-size: 24px;
}
```

### Rendering as Another Element

You can use the `as` prop to change the element type.

```tsx
import { Text } from '@devup-ui/react'

function App() {
  return (
    <Text as="p" color="red" fontSize="24px">
      This is Text component.
    </Text>
  )
}
```

The Text component defined above will render like this:

```tsx
function App() {
  return <p className="aa ab">This is Text component.</p>
}
```

```css
.aa {
  color: red;
}
.ab {
  font-size: 24px;
}
```
