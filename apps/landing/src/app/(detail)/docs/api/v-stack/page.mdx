# VStack

The `VStack` component is a layout component that uses flexbox to vertically layout its children.

It has a `display: flex` style with `flex-direction: column`.

## How to use

```tsx
const before = (
  <VStack>
    <Box bg="blue" h={25} w={25} />
    <Box bg="blue" h={25} w={25} />
    <Box bg="blue" h={25} w={25} />
  </VStack>
)

const after = (
  <div className="d3 d4">
    <div className="d0 d1 d2"></div>
    <div className="d0 d1 d2"></div>
    <div className="d0 d1 d2"></div>
  </div>
)
```
