# Image

The `Image` component is a layout primitive that can be used to create any kind of layout.

It is just a `img` with some styles.

## How to use

```jsx
// Before
<Image src={"https://via.placeholder.com/150"}/>

// After
<img src={"https://via.placeholder.com/150"}/>
```
