# Image

The `Image` component is a layout primitive that can be used to render images.

It is just a `img` with some styles.

## How to use

```tsx
import { Image } from '@devup-ui/react'

function App() {
  return <Image src="https://via.placeholder.com/150" />
}
```

The Image component defined above will render like this:

```tsx
function App() {
  return <img src="https://via.placeholder.com/150" />
}
```
